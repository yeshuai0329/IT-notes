# 前端组件设计

> ​		React 和 Vue 等框架中组件化都有所体现。组件通常是由标记语言、逻辑和样式组成的集合。它们被创建的目的就是作为可复用的模块去构建我们的应用程序。
>
> ​		组件是大多数现代前端框架的基本概念之一。组件设计，包括组件层级结构设计以及组件各自的职责的划分。
>
> ​		在设计组件的时候需要考虑到很多方面，以便它们可以很好的复用，组合，分离和解耦，功能可以比较稳定的实现。

我认为组件化设计应该要注意一下几个部分：

1. 层次结构。
2. 扁平化、面向数据的 state / props。
3. 更加纯粹的state变化
4. 解构，可复用、可组合
5. 配置化开发，可拓展
